# Basic CSS Website Portfolio

A clean, responsive, and minimalist two-page personal portfolio website built using vanilla **HTML5** and **CSS3**. This project showcases a neat structure for presenting personal projects, skills, and interests, alongside a functional contact layout.

---

## 🚀 Features

*   **Two-Page Navigation:** Smooth transition between the Home/Portfolio page and the Contact page.
*   **Portfolio Showcase:** Clean layout displaying featured clone projects (Amazon, Netflix, Todo App) with clear descriptions.
*   **Interests & Skills:** A dedicated structured section detailing core technical focus areas.
*   **Contact Form:** A beautifully structured tabular input form for user inquiries.
*   **Social Connectivity:** Integrated direct links to professional networks like LinkedIn.
